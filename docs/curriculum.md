   from transformers import AutoTokenizer, AutoModel

   - **答案：** B

2. **编程题：** 实现完整的训练循环
   ```python
   for epoch in range(num_epochs):
       for batch_x, batch_y in dataloader:

           outputs = model(batch_x)
           loss = criterion(outputs, batch_y)
           loss.backward()
           optimizer.step()
   ```
