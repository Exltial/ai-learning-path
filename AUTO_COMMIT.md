# 每小时自动提交代码并汇报进展

## 任务说明
每隔一小时：
1. 检查项目文件变更
2. git add .
3. git commit -m "Auto-commit: [时间戳] [进展摘要]"
4. git push
5. 向用户汇报本次提交的变更和整体进展

## 执行时间
每小时整点执行（如 13:00, 14:00, 15:00...）

## 汇报格式
```
🕐 [时间] 自动提交完成

📝 本次变更:
- [文件/功能 1]
- [文件/功能 2]

📊 整体进展:
- Phase 1: ✅ 完成
- Phase 2: 🔄 XX% 完成
- Phase 3: ⏳ 待开始

👥 团队状态:
- [Agent 状态]
```
